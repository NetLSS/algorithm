데이터의 개수 | 메모리 사용량 
---------|----------|

---------|----------|---------
 동작 원리 | 스택 | 큐

### 정의

- 탐색(Search): 많은 양의 데이터 중에서 원하는 데이터를 찾는 과정
- 자료구조(Data Structure): 데이터를 표현하고 관리하고 처리하기 위한 구조
- 정렬(Sort): 데이터를 특정한 기준에 따라서 순서대로 나열하는 것
